## 🚀 快速开始

```bash
# 本地预览（需要Ruby + Jekyll）
bundle install
bundle exec jekyll serve

# 或使用Docker
docker run --rm -p 4000:4000 -v $(pwd):/site bretfisher/jekyll-serve
```

访问 http://localhost:4000 查看博客。
